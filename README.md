# Quick-Calc

## Project Description
Quick-Calc is a simple Java calculator application that supports addition, subtraction, multiplication, division (with zero handling), and clear functionality. It demonstrates clean coding, unit and integration testing, and proper version control.

## Setup Instructions
1. Install Java 17+ and Maven
2. Clone repository
3. Navigate to project folder

## Running the Application
```bash
mvn compile exec:java -Dexec.mainClass="calculator.QuickCalcApp"
